# Industrial Fleet TCO & Transition Analysis

## Project Overview

This project performs a 5-year Total Cost of Ownership (TCO) analysis comparing Electric Vehicles (EVs) and Diesel commercial fleets using operational and financial business metrics.

The objective is to evaluate long-term fleet transition feasibility through lifecycle cost modeling, operational analytics, and business-driven data analysis.

---

## Business Problem

Commercial fleet operators face increasing pressure to:

* reduce operational cost
* improve sustainability
* optimize lifecycle asset utilization
* evaluate EV transition feasibility

This project analyzes the long-term financial impact of EV and diesel fleet operations using cost engineering and operational analytics techniques.

## Dataset Description

The dataset was created using publicly available commercial vehicle specifications and simulated operational fleet assumptions for EV and diesel vehicles.

The dataset contains:

* vehicle acquisition details
* fuel/electricity operational cost
* maintenance assumptions
* tyre cost assumptions
* government subsidy parameters
* resale value assumptions

---

## Key Analytical Components

### Data Loading & Reshaping

* SQL-exported CSV preprocessing
* relational data restructuring
* operational assumption extraction

### Feature Engineering

* fuel cost per kilometer calculation
* lifecycle maintenance estimation
* subsidy calculation
* resale value estimation

### TCO Modeling

The 5-year TCO model includes:

* acquisition cost
* operational fuel/electricity expense
* maintenance & tyre cost
* subsidy adjustments
* lifecycle resale value

### Data Visualization

Interactive visualizations were developed for:

* EV vs Diesel TCO comparison
* lifecycle cost component analysis
* feature importance analysis
* operational cost distribution

---

## Key Findings

* EV fleets demonstrated lower long-term operational cost.
* Fuel cost per kilometer was identified as a dominant TCO driver.
* EV operational savings offset higher acquisition cost over time.
* Government subsidies significantly improved EV transition feasibility.
* Lifecycle resale value reduced overall ownership cost.

---

## Business Recommendation

For high-mileage commercial fleet operations, EV adoption demonstrates strong long-term financial potential due to lower operational energy expenses and improved lifecycle cost efficiency.
